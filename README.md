# TMLP

This repository contains the code and data used for the TMLP project.

## Environment setup

We recommend using `conda` to create the environment.

```bash
cd TMLP
conda env create -f environment.yml
conda activate TMLP
```

## Special setup for `fzz`


```bash
cd fzz
python -m pip install -e .
cd ..
```


## Running the code

The persistence images (PIs) used in the experiments can be generated by running python time_zigzag.py for the Cross-Patch Zigzag PI and python ba_group_zigzag.py for the Intra-Patch Zigzag PI. The generated PIs have already been provided in the PI/ and PI_ba_groups/ folders.

After the PIs are prepared, a typical experiment can be launched from the project root using:
```bash
BA=CISO YEAR=2018 LR=0.001 L1=0.5 L2=0.5 dropout=0 bash scripts/run.sh
```
